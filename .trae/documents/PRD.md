## 1. 产品概述

《江湖·人情世故模拟器》是一款古风修仙题材的网页文字游戏 UI，以水墨丹青为视觉基底，文字叙事为主、状态栏与按钮为辅，营造沉浸式修仙氛围。

* 核心目的：为玩家提供境界修炼、功法研习、宝物收藏、画符炼丹、门派经营、社交往来的完整修仙体验入口

* 目标用户：喜爱修仙/仙侠题材、偏好文字 RPG 与古风美学的玩家

* 价值定位：以一页式精致面板整合六大子系统，让玩家在墨色山水中体悟修真百态

## 2. 核心功能

### 2.2 功能模块

1. **主界面**：顶部道号与境界横幅、左侧导航六系、主舞台展示当前所选系统、底部状态条（气血/灵力/神识）
2. **个人信息**：境界修为、灵根资质、寿元天命、体魄状态、修炼历程年表
3. **功法系统**：主修功法、辅修功法、功法熟练度进度、参悟心得、突破按钮
4. **财产与宝物**：灵石库存、储物袋格子、法宝装备槽、珍稀材料分类
5. **制作系统**：画符台（符纸/朱砂/符箓图谱）、炼丹炉（丹方/药引/火候控制）
6. **门派经营**：门派信息、职位晋升路径、门派任务榜、贡献度商店，门派底蕴和库存等
7. **社交系统**：道侣、师徒、好友、仇敌四类关系，亲疏进度与互动按钮

## 3. 核心流程

玩家进入主界面后，默认展示个人信息面板，可见自身境界与寿元。通过左侧导航切换至功法系统可参悟新功法；切换至制作系统可选择画符或炼丹进行道具生产；门派经营中可接取任务赚取贡献；社交系统中可与道侣双修、与好友往来。所有操作消耗或回复气血/灵力/神识，底部状态条实时反馈。

## 4. 用户界面设计

### 4.1 设计风格

* **主色调**：水墨黑 `#1a1612` 为底，宣纸米黄 `#e8dcc0` 为面，朱砂红 `#a83232` 为点睛，松烟墨青 `#3a4a4a` 为辅

* **强调色**：金箔色 `#c9a961` 用于境界与珍稀标识

* **按钮风格**：方形圆角印章式按钮，朱砂红底白字，悬停浮起带墨晕阴影

* **字体**：标题用 `Ma Shan Zheng`（马善政毛笔楷书），正文用 `Noto Serif SC`（思源宋体），数值用 `ZCOOL XiaoWei`

* **布局**：左侧固定导航 + 主舞台卡片式布局 + 底部固定状态条

* **装饰**：山水水墨背景、卷轴边框、印章纹饰、云纹分割线

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素                           |
| ---- | ---- | ------------------------------- |
| 主界面  | 整体布局 | 三栏式：左导航 220px、主舞台自适应、底部状态条 80px |
| 主界面  | 背景   | 水墨远山图层 + 宣纸纹理叠加 + 暗角晕染          |
| 个人信息 | 境界面板 | 九层境界竖向阶梯图，当前层金光高亮               |

| 功法系统 | 熟练度环 | SVG 圆环进度，中心显示百分比                |
| 财产宝物 | 储物袋  | 8x6 网格，每格圆角方块，物品图标+数量           |
| 制作系统 | 炼丹火候 | 横向滑块，三段火候标注文武火                  |

| 社交系统 | 关系卡片 | 头像位 + 姓名 + 亲疏进度条 + 互动按钮         |

### 4.3 响应式

桌面优先设计，最小宽度 1280px。在 1024px 以下时左侧导航收为图标条，主舞台保持单列。移动端暂不深度适配，保证桌面端完整体验。

### 4.4 3D 场景

本项目为 2D 文字游戏 UI，不涉及 3D 场景。
